## 디렉토리 구조

```
├── images
|   ├── common // 공통 이미지
|   |   ├── logo // 로고 이미지
|   |   └── ico // 아이콘 이미지 (ex. ico_auth)
|   └── pic // 기타 이미지 
|
├── js
|   ├── common // 공통 .js (ex. jquery)
|   └── lib // js 라이브러리 (ex. swiper)
|
├── styles
|   ├── common // 공통 css (ex. reset)
|   └── lib // 라이브러리 css (ex. swiper)
|
└── .gitignore // git 업로드시 제외되는 파일 지정
```
